# opengl


###  着色器语言

> 数据类型

>> 标量
*  int 有符号整形  取值  -32768~32767
*  uint  无符号整形 取值 0~65535
*  bool  布尔类型
*  float  浮点型

>> 向量  (是由标量组成的)
* vec2 2个浮点数的向量
* ivec3 3个有符号整形的向量
* uvec4 4个无符号整形的向量
* bvec4 4个布尔数的向量

>> 矩阵    (是由多个向量组成的)
* mat2  2×2的浮点数矩阵
* mat3  3×3的浮点数矩阵
* mat2×3,  2×4的浮点数矩阵
